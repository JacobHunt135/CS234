Instructions to run and compile.
- all files under this github directory are components of the program, excluding the project submission documents
- once files are aquired and placed into one directory, go into the directory of the project in the terminal
- next, run the command 'javac Main.java'. This will compile the program.
- next, run the command 'java Main' to run the program.

- NOTE: the program was predominantly developed in Visual Studio Code and, as such, is ran like a usual VSC program
