- As a .jar file, the Hardware_Store executable can be ran by simply double-clicking the file to start the program.
- Alternatively, the program can be ran from CMD by the command 'java -jar "Hardware_Store.jar"'.
- For proper read/write, the other .txt files here (besides this readme) must be placed under the same directory as the executable.
